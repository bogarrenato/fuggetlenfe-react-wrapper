# @fuggetlenfe/react-wrapper

React wrapper for `@fuggetlenfe/components` – auto-generated by Stencil React output target.

## Exports

- **`.`** — React component wrappers (e.g., `FfButton`)
- **`./server`** — `renderToString` / `hydrateDocument` for SSR with Declarative Shadow DOM

## Usage

```tsx
import { FfButton } from '@fuggetlenfe/react-wrapper';

function App() {
  return <FfButton variant="primary">Click me</FfButton>;
}
```
